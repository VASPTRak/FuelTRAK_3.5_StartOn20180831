Translation Table
The transaction button provides the user access to:
  • Search and or edit a transaction
  • Create a new transaction, or
  • Export transactions.
From the Home page, do the following:
  1. Hover over the Transaction button on the menu located on the left hand side of the screen.
  1. Select Export Transactions with a single click.
  1. Select the Translation Table  button with a single click.
Indicates that translation tables have been saved and are available for edit.
