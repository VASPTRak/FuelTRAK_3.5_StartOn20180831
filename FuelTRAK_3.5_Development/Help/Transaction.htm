Transaction Search/Edit
The transaction button provides the user access to:
  • Search and or edit a transaction,
  • Create a new transaction, or
  • Export transactions.
From the Home page, do the following:
  1. Hover over the Transactions button on the menu located on the left hand side of the screen.
  2. Select Search/Edit with a single click.
  3. Input search criteria as required.  The Date From and To fields are mandatory, all other search fields are optional.
  4. Click the Search button.
All optional fields can be run as a single item search or a range search by filling out the From and To fields.  When running a single item search it is necessary to place the

data in both the From and To fields. (Example: Department# - From 001 To 001)
  1. Date: Select the date range for the report.  When entering the date it must be formatted as mm/dd/yyyy; you can also use the popup calendar to populate the date fields. If all transactions are required for the report fill only the dates required and click the Search button.
  2. Vehicle ID:  Enter a range or a single Vehicle ID; maximum ten (10) character field. (Example: From 354 - will pull all transactions for vehicle 354 listed in date time order, From 354 To 356 will pull all transactions for vehicles 354, 355 and 356). 
  3. Sentry#: Enter a range or a single Sentry number; This is a numeric field with a maximum of three (3) digits. (Example: a Sentry number should be entered as a 3 digit number - 002)
  4. Key#: Enter a range or a single Vehicle Key number, this is a numeric field with a maximum of five (5) digits. (Example: a key number should be entered as a 5 digit number - 01017)
  5. Personnel#: Enter a range or a single Personnel number; maximum ten (10) character field.
  6. Veh. Model:  Enter the vehicle's model; maximum twenty (20) character field.
  7. Veh. Key:  Enter the key number assigned to the vehicle; this is a numeric field with a maximum of five (5) digits. (Example: a key number should be entered as a 5 digit number - 01017)
  8. VIN #:  Enter the Vehicle Identification Number; maximum of twenty (20) characters.
  9. Veh. Desc.:  Enter the vehicle's description; maximum of fifty (50) characters.
  10. Veh. Year:  Enter the vehicle's year; maximum of four (4) characters.
  11. Veh Make:  Enter the vehicle make. (Example: Honda, Chevy, Ford...)
  12. Veh. Card:  Enter the vehicle's assigned employee card number; maximum seven (7) characters.
  13. License #:  Enter the vehicle's license plate number; maximum nine (9) characters.
  14. Dept #: Enter a range or a single Department number; maximum three (3) digit field. (Example: 001).
The search results will be displayed below the search criteria fields.
Once the transaction search is completed the user can either Edit or Delete a transaction.

Select the Edit link to the left of the transaction to open the transaction in the edit screen.

  • Make any changes necessary.
  • Some fields are locked and cannot be changed:  Description, Department, Key #, Tank, Product and Cost.
    • Description
    		 The vehicle description can only be changed from the Vehicle Search/Edit screen.  Note, if you change the vehicle's description it will apply to all transactions performed by that vehicle.
    • Department
    		 The vehicle's assigned department can only be changed from the Vehicle Search/Edit screen.  Note, if you change the vehicle's description it will apply to all transactions performed by that vehicle.
    • Key #
    		 The key number cannot be changed.  It is automatically issued by WinCC when a key is encoded.
    • Tank
    		 The assigned tank can only be changed from the Sentry Search/Edit screen.  Note, if you change the Sentry's assigned tank it will apply to all transactions made at that Sentry.
    • Product
    		 The product can only be changed from the Tank Search/Edit screen.  Note, if you change the tank's assigned product(s) it will apply to all transactions made at that Sentry.
    • Cost
    		 The cost field is an auto calculate field, it cannot be changed.  If the quantity field is changed the cost field will recalculate accordingly.
  • The following fields can be changed without affecting other transactions.
    • Date
    		 Enter the date manually or use the calendar popup icon located to the right hand side of the date field.
    • Time
    		 Enter the date in 24-hour notation. (Example: 18:00 for 6:00pm)
    • Transaction No.
    		 The transaction number can be changed but it is recommended that when it is *MAN* that you do not change it.  *MAN* indicates that the transaction was a manual transaction.  Changing this field can corrupt your data and make information harder to find at a later date.
    • Sentry
    		 Use the pull down menu to change the Sentry. Changing the Sentry number will automatically repopulate the Tank and Product fields with those assigned to that Sentry.
    • Vehicle
    		 Use the pull down menu to change the Vehicle.  Changing the vehicle number will automatically repopulate the Description and Department fields with those assigned to that vehicle.
    • Personnel Name
    		 Use the button to the right of the field to open a pop up menu with a listing of personnel to choose from.
    • Optional
    		 The optional field can be used to add notes, data is optional in this field.  Maximum thirteen (13) character field.
    • Errors
    		 It is recommended that this field remain unchanged.  It indicates that this transaction had an error.
    • Odometer
    		 Delete the existing odometer and enter the new odometer reading.
    • Hours
    		 Delete the existing hours and enter the new hour reading.
    • Hose
    		 Use the pull down menu to change the hose number.
    • Quantity
    		 Delete the existing gallons and enter the new gallons, this will automatically be recalculated and change the cost field accordingly.
  • Select the Save button to save changes and return to the previous screen.
  • Select the Cancel button to exit without saving the changes.
