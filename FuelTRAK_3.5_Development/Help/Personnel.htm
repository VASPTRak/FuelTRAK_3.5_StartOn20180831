Personnel
Click on the Search / Edit option from the Personnel under User menu as shown below.
If you click on 'Search / Edit' submenu 'Personnel' screen will be displayed.
Click on the New button to display the following screen. Enter the required data into the respective fields and click on Ok button, this will add the current record to the database and the same window will appear again to enter another record else press the Cancel button.
Fields for New Personnel Information window:
Personnel Id :- Enter Personnel Id, alphanumeric value(maximum limit 10 digit)..
Key :- Enter key number.
Key Exp - Enter the Key Exp. Format for this field is mm/yyyy.
Card Exp :- Enter the Card Exp. Format for this field is mm/yyyy.
First Name :- Enter first name of person.
Department :- Select Department from the selection box.
MI :- Enter MI, numeric and alphanumeric value are accesped (maximum limit 1 digit).
Last Name :- Enter last name of person.
Ok Button :- Click this button to save the current record and exit.
Cancle Button :- Click this button to Exit from this window without saving the record.
Clicking on the Search button will display the window as shown below.
Fields for Personnel-Search Criteria window:
Personnel Id Field :- Enter the Personnel ID, alphanumeric values are accepted and the records will be searched according to Personnel Id.
Last NameField :- Enter the Last name, character values are accepted and the records will be searched according to Last Name.
Account Id Field :- This field will allow to enter Year in the format (yyyy) and the records will be searched according to that year.
Key NumberField :- Enter the key number, it will search for the match case from the available records,numeric value are accepted.
Card Number Field :- Enter the Card Number, values are accepted in format (mm/yyyy) and the records will be searched accordingentered criteria.
Department Field :- Enter the Department Name, and the records will be searched according to entered criteria.
Search Button :- After entering the search criteria e.g. date, year etc. Click on 'Search' button to display the record according the entered criteria.
Edit Link :- Click on this link to open the selected record for editing.
