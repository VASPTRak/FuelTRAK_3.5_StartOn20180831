Export Transactions
The transaction button provides the user access to:
  • Search and or edit a transaction
  • Create a new transaction, or
  • Export transactions.
From the Home page, do the following:
  1. Hover over the Transaction button on the menu located on the left hand side of the screen.
  2. Select Export Transactions with a single click.
　
  • Input the appropriate data into each of the fields.
    • Date
    		 Enter the date manually or use the calendar popup icon located to the right hand side of the date field.
    • Time
    		 Enter the date in 24-hour notation. (Example: 18:00 for 6:00pm)
    • Department
    		 Enter a range or a single Department number; maximum three (3) digit field. (Example: 001).
    • Export Format
    		 Select the export format required.  These are report options according to a translation table.
    • Include Previous
    		 By checking this box you include all transactions including those previously exported.
