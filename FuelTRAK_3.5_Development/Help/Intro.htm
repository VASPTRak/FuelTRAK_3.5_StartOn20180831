Introduction
Welcome to FuelTRAK's Help Menu. 

Open help pages for each item listed on the FuelTRAK Menu tree at the left of the screen by clicking on the item.  Help pages can also be opened by a single click on the links provided throughout each page.  A link is indicated as underlined navy text - Example.

Open FuelTRAK via your internet browser.  IT Support will issue all authorized personnel a username and password; they will also provide the web address (Example: http://www.mycompany.com/fueltrak/Loginpage.aspx).  Once FuelTRAK has loaded in the browser, as a security measure, you will be asked to enter your Login information.  You cannot enter FuelTRAK without a Username or Password.

　
Figure 1
Once you have successfully logged in, the Home page will load.  The Menu buttons located on the left hand side of the screen will be used to navigate in FuelTRAK.  The buttons function in different ways. Some buttons are clicked to activate, some are hovered over to access a drop down menu for more options, see Figure 3.  The menu buttons are stationary and will remain at the left hand side of the screen on every page load.

Menu Button Description
　 Single Click to activate.

This button will return you to the home page, see Figure 2.  The home page contains contact information for your convenience.  We invite all comments or questions you may have, simply select the link and drop us an email or you can contact us by phone during our business hours.

　
　 Single Click to activate. 

This button will open a screen with all of the options associated with running reports.  Reports are broken down into eight (8) categories, Transaction, Site, Billing, Vehicle, Personnel, Fuel Use, Inventory and Miscellaneous.  Each category contains a listing of available reports.  Changes to data cannot be performed in any of the form options, it is for printing and viewing data only.  Report Options are available for both printing or exporting run reports.

　
　 Hover to activate, select a menu option with a single click.

On hover this button will offer the option to: Search/Edit, New or Export Transactions. Dependent upon the Users security level transactions can be edited or created.  The Transaction Page allows the user to change information in a transaction, it also allows the user to add a Manual Entry transaction into the database. The Export function enables the user to export data to an outside source.

 
　 Hover to activate, select a menu option with a single click.

Inventory is split into three (3) categories: Tank Inventory, Meter Reading, and Tank Chart screens allow the user to Add, Search or Edit inventory records and entries.  In Tank Inventory you can add or edit Fuel Delivery, Tank Settings, and Tank Dippings.

　
　 Hover to activate, select a menu option with a single click.　

Item is broken down into eight (8) categories: Vehicle, Personnel, Department, Sentry, Product, Tank, Tank Monitor and Read Key.  Each category allows you to enter into that item and either edit or add data with the exception of Read Key.  In the Read Key screen there are no options available, it is for information purposes only. Changes or edits cannot be made, it will simply read either a vehicle or personnel key and provide you with the key's encoded data.
