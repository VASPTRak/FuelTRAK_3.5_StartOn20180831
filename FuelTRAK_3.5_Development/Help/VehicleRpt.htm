Vehicle Reports
From the Home page, do the following:
  1. Select Reports from the menu located on the left hand side of the screen.
  2. Select Vehicle by clicking on the applicable radio button.
  3. Highlight the Vehicle Report that you would like to run.
  4. Click the Next button.

All optional fields can be run as a single item search or a range search by filling out the From and To fields.  When running a single item search it is necessary to place the data in both the From and To fields. (Example: Department# - From 001 To 001)

Search Criteria Fields:
  1. Vehicle ID:  Enter a range or a single Vehicle ID; maximum ten (10) character field. (Example: From 354 - will pull all transactions for vehicle 354 listed in date time order, From 354 To 356 will pull all transactions for vehicles 354, 355 and 356). 
  2. Department #: Enter a range or a single Department number; maximum three (3) digit field. (Example: 001).
  3. Vehicle Type: Enter a range or a single Vehicle Type; this field accepts a maximum of two (2) characters. (Example: From 2D To 4D)
  4. Vehicle Key#: Enter a range or a single Vehicle Key number, this is a numeric field with a maximum of five (5) numerals. (Example: a key number should be entered as a 5 digit number - 01017)
  5. Include Deleted Vehicles: Check this box if you want to run a report that includes deleted Vehicles.
