Department
Click on the Search / Edit option from the Department under User menu as shown below.
If you click on 'Search / Edit' submenu 'Department' screen will be displayed.
Click on the New button to display the following screen. Enter the required data into the respective fields and click on Ok button, this will add the current record to the database and the same window will appear again to enter another record else press the Cancel button.
Fields for New Department Information window:
Department Number :- This field is auto incremented and readonly.
Name :- Enter Name of Dapartment.
Address :- Enter address of department.
Surcharge :- Enter surcharge of department. Format for this field is 0.000, only numeric values are allowed.
Account Number :- Enter account number (numeric and alphanumeric values are accepted).
Upload Code :- Enter upload code of department.
Add Button :- This button is used to save the current record and will allow to enter another record.
Save Button :- This field will allow to save/Update record and will shown same window.
Cancle Button :- Click this button to Exit from this window without saving the record.
Clicking on the Search button will display the window as shown below.
Fields for Department-Search Criteria window:
Department Name Field :- Enter the Department Name, and the records will be searched according to Department name.
Department Number Field :- Enter the Department Number, numeric values are accepted and the records will be searched according to Department Number.
Search Button :- Click this button to display the record according the entered criteria.
Edit Link :- Click on this link to open the selected record for editing.
