Billing Reports
From the Home page, do the following:
  1. Select Reports from the menu located on the left hand side of the screen.
  2. Select Billing by clicking on the applicable radio button.
  3. Highlight the Billing Report that you would like to run.
  4. Click the Next button.
You will be prompted to enter report search criteria; The date fields are mandatory on all searches, all other fields are optional.

All optional field can be run as a single item search or a range search by filling out the From and To fields.  When running a single item search it is necessary to place the data in both the From and To fields. (Example: Department# - From 001 To 001)

Search Criteria Fields:
  1. Date: Select the date range for the report.  When entering the date it must be formatted as mm/dd/yyyy; you can also use the popup calendar to populate the date fields.  If all transactions are required for the report fill only the dates required and Click Run Report.
  2. Time: Enter the date in 24-hour notation. (Example: 22:00)
  3. Vehicle ID:  Enter a range or a single Vehicle ID; maximum ten (10) character field. (Example: From 354 - will pull all transactions for vehicle 354 listed in date time order, From 354 To 356 will pull all transactions for vehicles 354, 355 and 356). 
  4. Department #: Enter a range or a single Department number; maximum three (3) digit field. (Example: 001).
  5. Tank #: Enter a range or a single Tank number; maximum three (3) digit field.
  6. Vehicle Type: Enter a range or a single Vehicle Type; this field accepts a maximum of two (2) characters. (Example: From 2D To 4D)
  7. Vehicle Key#: Enter a range or a single Vehicle Key number, this is a numeric field with a maximum of five (5) numerals. (Example: a key number should be entered as a 5 digit number - 01017)
  8. Sentry#: Enter a range or a single Sentry number; This is a numeric field with a maximum of three (3) numerals. (Example: a Sentry number should be entered as a 3 digit number - 002)
  9. Personnel ID: Enter a range or a single Personnel number; maximum ten (10) character field.
  10. Zero Quantity Transactions Only: Check this box if you want to run a report with all zero transactions only.
　
Each report will provide a different set of Search Criteria options.
