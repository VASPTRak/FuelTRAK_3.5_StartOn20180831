Create New User
From the Home page, do the following:
  1. Select Setup from the menu located on the left hand side of the screen.
  2. Select User.
  3. Select New.
The following screen will open, input the data required for the new user and select Create.
Fields for New User Window:
User Name: Enter the User Name e.g. admin (numeric values are not allowed).
Password: Enter the Password.
First Name: Enter First Name for the new user (numeric values are not allowed).
Last Name: Enter Last Name for the new user (numeric values are not allowed).
Level : Select Level of security for the new user (both User & Admin levels have access to this screen, they have authorization to add a new user).
Create Button : Select this button to save the new user.
Cancel Button : Select cancel to Exit from this window.
