Miscellaneous Reports
From the Home page, do the following:
  1. Select Reports from the menu located on the left hand side of the screen.
  2. Select Misc. Reports by clicking on the applicable radio button.
  3. Highlight the Vehicle Report that you would like to run.
  4. Click the Next button.
　
Listing - Departments This is an automatically run report.  Once you select next the report will run, there are no search criteria options available for this report.

Produces a report of all departments currently in the database, listed in ascending order by department number.

　
Listing - Site Information This is an automatically run report.  Once you select next the report will run, there are no search criteria options available for this report.

Produces a report of all sites currently in the database, grouped by sentry number then listed in ascending order by hose number.

　
Listing - Tanks This is an automatically run report.  Once you select next the report will run, there are no search criteria options available for this report.

Produces a report of all tanks currently in the database, listed in ascending order by tank number.

　
Listing - Lockouts This is an automatically run report.  Once you select next the report will run, there are no search criteria options available for this report.

Produces a report of all key lockouts currently in the database.

　
Listing - Current Export File This is an automatically run report.  Once you select next the report will run, there are no search criteria options available for this report.

Produces a transaction report previously exported, grouped by sentry number then listed in ascending order by date.
