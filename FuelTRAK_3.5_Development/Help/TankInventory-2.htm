Tank Inventory
Inventory has two options as 'Inventory Entries' and 'Meter Readings'. Click on Search / Edit for Tank Inventory to search or edit the records from Tank Inventory. Click on Search / Edit for Meter Readings to search or edit the records from Meter Readings.
Selecting the Search / Edit for Tank Inventory will display the screen as below.
Click 'New' button to add new record to Tank Inventory. Clicking the new button will display popup as shown below. It has three types as 'Fuel Delivery', 'Tank Setting' and 'Tank Dipping'. Select the type according to your requirements to add new record.
e.g. 1) Selecting the Fuel Delivery will display the screen as shown below to add the new record as Fuel Delivery to Tank Inventory

Fields for Tank Delivery:
Tank: Select the appropriate tank from selection box.
Level Prior to Delivery: Enter the numeric value maximum limit is 99999 (5 digit).
Quantity Delivered: Enter the numeric value maximum limit is 99999 (5 digit).
Delivered Fuel Price: Enter the numeric value maximum limit is 9.99999 (5 decimal).
Time of Delivery: Enter the Time of Delivery. Format for this field is hh:mm (e.g. 20:30) 24 Hour format.
Delivery: Enter the Date for Delivery. Format for the date is mm/dd/yyy (e.g. 10/20/2006).
Invoice#: Enter the Invoice number. Format for this field is alphanumeric. Combination of numbers and alphabets can be entered here.
Supplier: Enter the name of the Supplier. Format for this field is alphanumeric. Combination of numbers and alphabets can be entered here.
Employee: Enter the name of the Employee. Format for this field is alphanumeric. Combination of numbers and alphabets can be entered here.
2) Selecting the Tank Setting will display the screen as shown below to add the new record as Tank Setting to Tank Inventory.
Fields for Tank Setting:
Tank:
Select the appropriate tank from selection box.
Quantity:
Enter the numeric value maximum limit is 99999 (5 digit).
Price:
Enter the numeric value maximum limit is 9.99999 (5 decimal).
Time:
Enter the Time. Format for this field is hh:mm (e.g. 20:30) 24 Hour format.
Date:
Enter the Date. Format for the date is mm/dd/yyyy (e.g. 12/12/2006).
3) Selecting the Tank Dipping will display the screen as shown below to add the new record as Tank Dipping to Tank Inventory.
Fields for Tank Dipping:
Tank:
Select the appropriate tank from selection box.
Quantity:
Enter the numeric value maximum limit is 99999 (5 digit).
Time:
Enter the Time. Format for this field is hh:mm (e.g. 20:30) 24 Hour format.
Date:
Enter the Date. Format for the date is mm/dd/yyyy (e.g. 12/12/2006).
Buttons for Tank Dipping,Tank Settings, and Fuel Delivery :
Add Button:- Fill all the values to the fields and click on this button to add the record, this will add the record to database and brings the blank window to add another record.
Save Button:- Clicking on this button will save / update the changes to database and displays the same window with the changed values.
Cancel Button:- Click this button to exit from window without saving the changes.
Clicking on the Search button will display the window as shown below.
Search Criteria for Tank Inventory:
Date :- Enter the date. Format (mm/dd/yyyy) it will search the record according to the entered date.
Tank:- Enter the tank number in this field to search according to tank number.
Type:- Enter type of tank (e.g. Tank Setting, Tank Dipping or Fuel Delivery) to search according to tank type.
Edit Link:- Click this link to open the selected record for edit.
Search Button:- After entering the search criteria e.g. date, Tank etc. Click on 'Search' button to display the records according the entered criteria.
