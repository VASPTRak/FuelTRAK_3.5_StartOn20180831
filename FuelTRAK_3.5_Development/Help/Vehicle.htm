Vehicle
Click on the Search / Edit option from the Vehicle under User menu as shown below.
Click on 'Search / Edit' option to display the screen as shown below.
Click on the New button to display the following screen. Enter the required data into the respective fields and click on OK button. This will add the current record to the database and the same window will appear again to enter another record else press the Cancel button.
Fields for New Vehicle Information window:
Vehicle Id:- Enter Vehicle Id, alphanumeric value (maximum limit 10 digit).
License:- Enter License number, alphanumeric value (maximum limit 9 digit).
Key Exp- Enter the Key Exp. Format for this field is mm/yyyy.
Card Exp:- Enter the Card Exp. Format for this field is mm/yyyy.
Year:- Enter the Year. Format for this field is yyyy.
Department:- Select Department from the selection box.
Products- Click button next to Product, it will display the popup to select Products types.
PM Button :- Click button to enter the Preventive Maintainence data.
OK Button :- Click this button to save the current record and exit.
Cancel Button :- Click this button to Exit from this window without saving the record.
Click on the Search button to display the screen as shown below.
Fields for Vehicle-Search Criteria window:
Vehicle Id:- Enter the vehicle Id, alphanumeric values are accepted and the records will be searched according to Vehicle Id.
License No:- Enter the License No, alphanumeric values are accepted and the records will be searched according to License No.
Year:- This field will allow to enter Year in the format (yyyy) and the records will be searched according to that year.
Department:- Enter the Department , name or any letter from the depertment name, it will search for the match case from the available records.
Key Number:- Enter the Key Number, alphanumer values are accepted and the records will be searched according to Key Number.
Card Number:- Enter the Card Number, values are accepted in format (mm/yyyy) and the records will be searched accordingly.
Search Button:- Click this button to display the record according the entered criteria.
Edit Link:- Click on this link to open the selected record for editing.
Note :- If you want to search the record according to the fields (Vehicle Model, Vehicle Make, Description, VIN Number) then click on that respective button and enter the value in the textbox and click on search button, it will display the searched records.
