Meter Reading
Inventory has two options as 'Inventory Entries' and 'Meter Readings'. Click on Search / Edit for Meter Readings to search or edit the records from Meter Readings.
Selecting the Search / Edit for Meter Readings will display the screen as below.
Click 'New' button to add new record to Meter Readings. Clicking on the New button will display the following window. Enter the required data into the respective fields and click on OK button, doing this will add the current record to the database and the same window will appear again to enter another record else press the Cancel button.
Fields for New Meter Information window:
Sentry
Select sentry from the selection box.
Hose
Select the Hose Id from the selection box.
Reading
Enter the Reading. This is the numeric field. Format for this field is 0.00 and maximum limit is 9.99.
Time
Enter the Time. Format for this field is hh/mm.
Date
Enter the Date. Format for this field is mm/dd/yyyy.
Ok Button
Click this button to save the current record and exit from this window.
Cancel Button
Click this button to Exit.
Clicking on the Search button will display the window as shown below.
Search Criteria for Meter Reading:
Sentry:- Enter the Sentry, the records will be searched according to Sentry.
Date:- Enter the Date. Format for this field is mm/dd/yyyy and the records will be searched according to Date.
Search Button:- Click this button to display the record according the entered criteria.
Edit Link:- Click on this link to open the selected record for editing.
